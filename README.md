# orchestration
orchestration

# Usage

Install express-generator

```bash
$ sudo npm install express-generator -g
```

Gerar Modulo

```bash
$ express modules/mymodule
```

NPM Install

```bash
$ npm install
```

Run

```bash
$ MODULES=$(pwd)/modules/ npm start
```

Open

[Clique aqui](http://localhost:3000/mymodule)
